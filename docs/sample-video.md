# Sample Video

The repository includes `assets/footage/sample-road-360p.mp4`.

Source: https://samplelib.com/sample-mp4.html

Samplelib describes its MP4 files as available with no license restrictions. The downloaded file is the 360p road sample, about 1.1 MB.

Local metadata:

- Codec: H.264
- Width: 640
- Height: 360
- Duration: 5.7 seconds

The demo treats this single MP4 as several logical clips so the prototype can exercise prompt planning, role ordering, timeline construction, review scoring, and export without depending on private source footage.
